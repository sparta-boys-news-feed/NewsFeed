## 프로젝트 주요 기능

<details>
  <summary>**회원가입**</summary>
  - 이메일 중복 확인을 통해 유효한 이메일 주소가 있으면 회원 가입이 가능
  - 탈퇴 했던 이메일은 재사용 불가능
</details>
<details>
  <summary>**로그인**</summary>
  - 회원가입 당시 사용했던 이메일과 비밀번호로 로그인 가능
  - JWT 토큰을 기반으로 관리
</details>
<details>
  <summary>**게시글 작성**</summary>
  - 게시글 작성 가능
</details>

  <summary>**댓글 작성**</summary>
  - 게시글에 댓글 작성 가능
